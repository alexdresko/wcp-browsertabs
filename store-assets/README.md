# Store Listing Media

Prepared uploadable assets:

- `store-logo-300x300.png` - upload as the 1:1 app tile icon.
- `super-hero-1920x1080.png` - optional Store promotional artwork.

Screenshots still need to be captured from the real app UI before submission.

Screenshot requirements for Desktop:

- PNG format.
- 1366 x 768 pixels or larger.
- Less than 50 MB each.
- At least one screenshot is required.
- Four or more screenshots are recommended.

Recommended screenshot set:

- `01-command-palette-browser-tabs-results.png`
- `02-filter-open-tabs-by-title.png`
- `03-switch-to-selected-tab.png`
- `04-empty-state-or-extension-entry.png`

Capture actual Command Palette screens rather than mockups. Avoid extra marketing text, publisher logos, or decorative overlays in screenshots.
